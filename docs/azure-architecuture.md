# Azure構成

## Naming rules

`{resourcePrefix}-{baseName}-{randomString}`

- `{resourcePrefix}`: see [abbreviations.json](../infra/abbreviations.json)
- `{baseName}`: see [main.bicepparam](../infra/main.bicepparam)
- `{randomString}`: 4 characters unique string generated by Bicep

### Custom domains

- Azure DNS: specify by Bicep parameter `customDomainZoneName` and `customDomainName`.
- Key Vault: specify by Bicep parameter `keyVaultName` and `keyVaultSecretName`.
- Web App: `{suffix}.{customDomainName}`

## Log Analytics

All web apps use the same Log Analytics workspace and Application Insights resource.
